# Classification du Potentiel d'Exportation - Maroc 🇲🇦

Ce projet vise à identifier les produits marocains à fort potentiel d'exportation en utilisant des techniques de **Machine Learning**.

## Méthodologie
- **EDA :** Analyse descriptive et matrices de corrélation.
- **Preprocessing :** Standardisation des données et réduction de dimension (PCA).
- **Clustering :** Utilisation de l'algorithme **K-means** pour segmenter les produits en 3 groupes (Fort, Moyen, Faible potentiel).

## Contenu du dépôt
- `Rapport_Final.pdf` : Analyse détaillée et recommandations stratégiques.
- `Analyse_Technique.ipynb` : Code source Python (Scikit-Learn, Pandas, Seaborn).
